HTML Modals

HTML modals are modal popups that are displayed within the browser.

The Angular Template ships with 4 commonly-used templates that are available through the popupService.

Popup Windows

You can use the popup service to open a browser window or tab to any web site.

Note: Whether a window or tab is opened depends on the specific browser and how it is set.

The framework also allows browser popups to accept inputs. This only applies to pages under the same domain.

In the popup window, the input value is available in the shell controller through a variable called input.
